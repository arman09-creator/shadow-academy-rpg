# shadow-academy-rpg
RPG-based EdTech platform where students learn STEM through survival gameplay and missions.
## MVP Plan

Goal: Build a small playable demo where students solve STEM questions to progress in a survival RPG.

Core Game Loop:
1. Player enters a zone
2. Player solves a STEM question
3. Correct answer → Gain XP
4. XP unlocks next zone or item

## Tech Stack
- Game Engine: Unity
- Language: C#
- Database: Firebase
- Design: Figma

- Create question popup system
